reassocfunction in cjd.util.misc-utils

(reassoc [m k f])

(reassoc [m k f & args])

Returns a map that replaces a specified entry's value with a new value, computed as a function of the existing value, or of the existing value and some additional arguments.

Arguments

m — The map to be updated.

k — The key designating the value to update.

f — A function to update the value designated by key. f should have one argument if args are not supplied, or one plus the number of values in args if they are.

Returns

The updated map.

remapfunction in cjd.util.misc-utils

(remap [map-fn map])

(remap [map-fn map init-map])

Performs what is in effect a map-style operation on the entries of a map, resulting in a new map with entries transformed per a mapping function.

Arguments

map-fn — A mapping function, of the form (fn [key val]), where:

Arguments

key — A map entry's key

val — A map entry's value

Returns

A tuple of the form [k* v*], where

Arguments

k* — the mapped key

v* — the mapped value.

map — The map to be remapped.

init-map — An initial map to which remapped entries are assoc-ed. Optional; the default value is {}.

Returns

The remapped map.
