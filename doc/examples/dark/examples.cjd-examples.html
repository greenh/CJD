filtered-minmaxfunction in examples.cjd-examples

(filtered‑minmax [coll filter‑fn])

Returns the min and max values in a collection of numbers subject to a filtering constraint.

Arguments

coll — The collection to process.

filter‑fn — A filtering function of the form [value], where

value — The value to filter

Returns — True if the value is to be included in the min-max calculation.

Returns

A tuple of the form [min‑val max‑val], where:

min‑val — The minimum value of coll after filtering.

max‑val — The maximum post-filtering value of coll.

Note that min-val and max-val will be null if coll is empty after filtering.
