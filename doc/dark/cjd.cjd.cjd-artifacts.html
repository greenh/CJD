cjd.cjd.cjd-artifacts

Extends the core CJD artifact model to include a few additional artifacts, including extensos, constructos, and, of course, artifacts themselves.

parse-method-implementationsdefn in cjd.cjd.cjd-artifacts

(parse‑method‑implementations [parent forms])

Parses a sequence of method implementations sans any pooio declaration, as is found in the local methods of an extenso or constructo.

Note that parse-method-implementations directly updates the method implementation list in parent!

Arguments

parent — The parent artifact object, which must support the HasMethodImplementations protocol.

forms — The sequence of forms to parse from.

Returns

A sequence of unparsed forms at the tail of forms.
