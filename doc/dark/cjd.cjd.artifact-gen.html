cjd.cjd.artifact-gen

Adds support for CJD-specific artifacts to the base CJD HTML generator.

This is an extension to the base CDJ state of affairs, and is intended to be loaded by the "--requires" option (command-line version) or using the :requires option cjd-gen.

(gen‑extenso‑tree [extenso uninit‑fields context])

Generates a tree of extensos. This does a preorder traversal of the extenso tree rooted at extenso, outputting useful stuff as it goes.

Arguments

extenso — A var denoting the extenso to expand.

uninit‑fields — A set of symbols of names of uninitialized fields of the top-level extenso, used for decorative purposes.

context — Current context.

Returns

A string of the resulting HTML.
